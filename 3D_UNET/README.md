## UNET 3D

* Keras implementation

It is a keras implementation of [this](https://github.com/96imranahmed/3D-Unet) repository. <br>
For details, check `3D_UNET.ipynb`.


### Example Run
```python
unet3d = Model(inputs=model_input, outputs=conv_out, name='unet3d')
```
